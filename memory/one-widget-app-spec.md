# One Widget Life — 기획 설계서 (Claude Design용)

## 프로젝트 개요
ㅁ
**앱 이름 후보:** One Widget Life / 위젯하나 / MyPage / 원페이지
**한 줄 설명:** 앱 30개 대신 위젯 하나로 사는 앱
**핵심 가치:** 사용자가 원하는 정보와 기능만, 원하는 배치로, 하나의 위젯에서 해결

### 포지셔닝
```
Widgetsmith = 예쁘지만 기능 없음
Niagara     = 정리만 하지 대체는 아님
슈퍼앱      = 회사가 정한 메뉴
────────────────────────────────
이 앱       = 사용자가 직접 조립하는 나만의 위젯 슈퍼앱
```

## 타겟 유저

### 페르소나 1 — 미니멀리스트 직장인 (28세)
- 폰에 앱 47개인데 매일 쓰는 건 7개
- 홈화면 깔끔하게 쓰고 싶은데 위젯이 난잡함
- 아침에 뉴스+날씨+일정 한 눈에 보고 싶음

### 페르소나 2 — 반려동물 집사 (32세)
- 산책앱, 사료앱, 커뮤니티앱 따로따로 열기 귀찮음
- 산책 기록 + 사료 알림 + 동네 산책 친구 한곳에서 보고 싶음

### 페르소나 3 — 정보 수집가 (25세)
- IT뉴스, 주식, 커뮤니티 글 여러 소스에서 수집
- 네이버 들어가면 관심없는 피드부터 나와서 짜증남
- 내가 고른 소스만 깔끔하게 보고 싶음

## 화면 구조 (Information Architecture)

```
[홈 위젯] ─── 스와이프로 모듈 전환
    │
    ├── 모듈 1: 뉴스 피드
    ├── 모듈 2: 오늘 날씨
    ├── 모듈 3: 할 일 목록
    ├── 모듈 4: 강아지 산책
    ├── 모듈 5: 관심 주식
    ├── 모듈 6: 메모
    └── 모듈 N: (사용자 추가)

[앱 열기] ─── 위젯 탭 시 전체 화면
    │
    ├── 메인 대시보드 (모든 모듈 한눈에)
    ├── 개별 모듈 상세 뷰
    ├── 모듈 스토어 (추가/삭제/순서변경)
    └── 설정
```

## 핵심 화면 설계 (Wireframe + High Fidelity 요청)

### Screen 1: 홈 위젯 (Home Screen Widget)

**사이즈:** iOS Large Widget (전체 가로, 4×4) / Android 4×3~4×4
**레이아웃:**
```
┌─────────────────────────────────┐
│  ● ● ● ● ○ ○  (모듈 인디케이터) │
│                                 │
│  ┌─────────────────────────┐   │
│  │                         │   │
│  │   [현재 활성 모듈 콘텐츠]  │   │
│  │                         │   │
│  │   예: IT뉴스 3개 헤드라인  │   │
│  │   · "Claude Opus 4.7 출시"│   │
│  │   · "Apple WWDC 2026"    │   │
│  │   · "React 22 릴리스"     │   │
│  │                         │   │
│  └─────────────────────────┘   │
│                                 │
│  ← 스와이프로 다음 모듈 →        │
└─────────────────────────────────┘
```

**인터랙션:**
- 좌우 스와이프: 다음/이전 모듈
- 탭: 앱 전체 화면으로 진입
- 롱프레스: 빠른 모듈 전환 메뉴

**디자인 방향:**
- 다크 모드 기본 (OLED 배경과 자연스럽게 녹아들게)
- 라이트 모드 지원
- 모서리 라운딩 24px
- 글래스모피즘 배경 (반투명 + 블러)
- 폰트: SF Pro (iOS) / a2z (한글)
- 최소 폰트 14px, 제목 18px Bold

### Screen 2: 메인 대시보드 (Full App — Main)

**진입:** 위젯 탭 시 열림
**레이아웃:**
```
┌─────────────────────────────────┐
│  ☰  나의 페이지          ⚙️ 🔔  │
│─────────────────────────────────│
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │ 📰 뉴스   │ │ 🌤 날씨   │     │
│  │ 헤드라인3개│ │ 서울 23°  │     │
│  │          │ │ 맑음      │     │
│  └──────────┘ └──────────┘     │
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │ ✅ 할 일   │ │ 🐕 산책   │     │
│  │ 3개 남음  │ │ 오늘 0회  │     │
│  │          │ │ 🔴 산책가자│     │
│  └──────────┘ └──────────┘     │
│                                 │
│  ┌──────────┐ ┌──────────┐     │
│  │ 📈 주식   │ │ 📝 메모   │     │
│  │ AAPL +2.3%│ │ 최근 메모  │     │
│  │ TSLA -1.1%│ │ "장보기.."│     │
│  └──────────┘ └──────────┘     │
│                                 │
│  ┌─────────────────────────┐   │
│  │  ＋ 모듈 추가하기         │   │
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

**인터랙션:**
- 각 모듈 카드 탭 → 상세 뷰
- 각 모듈 카드 롱프레스 → 드래그로 순서 변경
- "+ 모듈 추가하기" → 모듈 스토어
- 카드는 2열 그리드, 스크롤 가능
- 카드 크기: 1×1 (작은), 2×1 (가로 넓은), 2×2 (대형) 사용자 선택

**디자인 방향:**
- 카드 기반 UI (각 모듈 = 하나의 카드)
- 카드 배경: 글래스모피즘 (rgba 반투명 + backdrop-blur)
- 카드 간 간격 12px
- 부드러운 그림자 (0 4px 16px rgba(0,0,0,0.15))
- 각 카드에 아이콘 + 모듈명 + 핵심 정보 요약
- 상태 표시: 초록(좋음), 주황(주의), 빨강(긴급)

## Claude Design 프롬프트

아래 프롬프트를 Claude Design에 그대로 입력:

### Wireframe 프롬프트:
```
모바일 앱 와이어프레임을 만들어줘.

앱 컨셉: "위젯 하나로 사는 앱" — 사용자가 원하는 정보 모듈(뉴스, 날씨, 할일, 강아지 산책, 주식, 메모 등)을 직접 골라서 하나의 위젯과 앱에 조합하는 커스텀 대시보드 앱.

필요한 화면 6개:
1. 홈 위젯 (iOS Large Widget) — 스와이프로 모듈 전환, 인디케이터 점
2. 메인 대시보드 — 2열 카드 그리드, 각 모듈 요약, + 추가 버튼
3. 뉴스 모듈 상세 — 카테고리 탭 + 기사 리스트
4. 강아지 산책 모듈 상세 — 산책 시작 버튼 + 주간 기록 + GPS 기록
5. 모듈 스토어 — 내 모듈 관리(드래그 순서변경) + 카테고리별 추가 가능 모듈
6. 온보딩 — 3스텝 (소개 → 모듈 선택 → 소스 설정)

스타일: 미니멀, 깔끔, 와이어프레임(회색톤), 아이콘+텍스트
```

### High-Fidelity 프롬프트:
```
위 와이어프레임을 기반으로 High-Fidelity 디자인을 만들어줘.

디자인 방향:
- 다크 모드 기본 (배경 #0A0A0F)
- 글래스모피즘 카드 (반투명 + backdrop-blur 20px)
- 디폴트 테마 컬러: 민트 #2DD4BF (사용자가 설정에서 자유롭게 변경 가능)
- 테마 프리셋: 민트 #2DD4BF / 블루 #60A5FA / 그린 #00E676 / 오렌지 #FBBF24 / 핑크 #F472B6 / 커스텀
- 디자인 시안은 디폴트 민트 테마로 만들어줘
- 카드 radius 16px, 부드러운 그림자
- 한글 폰트: a2z, 영문: SF Pro
- 각 모듈 카드에 컬러 아이콘 사용
- 온보딩은 일러스트 포함
- 위젯은 OLED 배경에 녹아드는 느낌
- 전체적으로 2026년 트렌드 반영: Neo-Dark, 미세한 그라디언트, 부드러운 모션 힌트

아이폰 15 Pro 프레임 (393×852) 기준으로 6개 화면 모두 만들어줘.
화면마다 실제 콘텐츠 예시 데이터를 넣어줘 (뉴스 헤드라인, 날씨 23°, 주식 AAPL +2.3% 등).
```

## 기술 스택 (참고)

### MVP (최소 버전)
- **프레임워크:** React Native (Expo) 또는 Flutter
- **위젯:** iOS WidgetKit (Swift) / Android Glance (Kotlin)
- **데이터:** 뉴스 RSS/API, 날씨 OpenWeather API, 주식 API
- **자체 기능:** 로컬 저장 (SQLite/AsyncStorage)
- **상태관리:** Zustand 또는 Riverpod

### 확장
- 서버: Supabase 또는 Firebase (동기화, 백업)
- AI: 뉴스 요약, 개인화 추천 (Gemini/Claude API)
- 소셜: 모듈 공유 (내 구성을 친구에게 공유)
